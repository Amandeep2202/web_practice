# HEADINGS & PARAGRAPHS

  • These tags (h1>h2>h3>h4>h5>h6) is used to make heading in this order.
  • <p> - This is a paragraph tag.
  • P*4 - This is used to write 4 paragraphs at once.
  • Lorem30 - This tag is used to write 30 words at once.
  • <strong> - This tag is used to bold the written text.
  • <em> - This tag is used to italicize the text.
  • <br> - It is a self closing tag which is used to break the line.
  • <hr> - It is also a self closing tag which is used to draw a horinzontal line.

# IMAGE & ANCHOR TAGS

  • ANCHOR TAGS :- It basically helps to handle links.
    1. href :- It is the attribute of anchor tag where you write the URL of any website.
    2. Keyword :- It is used by user and then, it redirect to the linked website.
      Example :- <a href="https://google.com">Go to Google</a>
    3. To open link in a new tab, you can use -
      <a href="https://google.com" target="blank">Go to Google</a>
  • IMAGE TAGS :- This tag is used to insert image.
    1. src :- This attribute is the field where you have to insert the URL or address of image.
    2. alt:- This attribute is a field that will display to users if their browser fails to load the image.
      3. Example :- <img src="https://source.unsplash.com/random" alt="Error loading image">
    3. We can also change the height and width of image using HTML.
      4. Example :- <img src="https://source.unsplash.com/random" alt="Error loading image"> height="200px" width="400px"

# LISTS & TABLE TAGS

  • LIST TAGS :- There are basically two types of Lists.
    1. Ordered Lists (<ol>)
      • <ol type="1"></ol>
      • Example:- 1,2,3,.... ; a,b,c,.... ; I,II,III,....; etc.
    2. Unordered Lists (<ul>)
      • <ul type="square"></ul>
      • Example:- disc; square; circle; etc.
  • TABLE TAGS :- It is just a combination of rows and columns on a webpage.
    1. <table></table>
